# MM Construções

Landing page simples desenvolvida para uma loja de materiais de construção.

O **MM Construções** é um projeto criado com fins de **aprendizado em desenvolvimento front-end**.  
A página simula o site institucional de uma loja de materiais de construção da minha rua, que atualmente não possui presença online.

O objetivo foi criar uma **landing page simples, organizada e funcional**, utilizando apenas tecnologias básicas da web.

---

## Sobre o projeto

Este projeto representa apenas a **página inicial do site**, contendo as principais seções que um site simples de comércio local normalmente possui.

A ideia foi criar uma estrutura clara e intuitiva para visitantes que desejam:

- Conhecer a loja
- Ver alguns produtos disponíveis
- Ler avaliações de clientes
- Entrar em contato com a loja

---

## Seções da página

A landing page contém as seguintes partes:

- **Header**  
  Área superior com o nome da loja e navegação entre as seções.

- **Início**  
  Apresentação da loja e mensagem principal.

- **Produtos**  
  Exibição de alguns materiais de construção oferecidos.

- **Mensagem dos compradores**  
  Depoimentos de clientes sobre a loja.

- **Contato**  
  Informações para entrar em contato com o estabelecimento.

---

## Tecnologias utilizadas

O projeto foi desenvolvido utilizando apenas tecnologias básicas:

- **HTML5**
- **CSS3**

⚠️ Nenhum **JavaScript** foi utilizado, pois o objetivo foi praticar **estruturação e estilização de páginas**.

---

## Objetivo do projeto

Este projeto foi desenvolvido para:

- Praticar **HTML semântico**
- Melhorar habilidades em **CSS**
- Criar layouts simples para negócios locais
- Desenvolver projetos para **portfólio**

---

## Status do projeto

Projeto criado apenas para **fins de estudo**.

Atualmente contém:

- Landing page completa
- Estrutura das seções principais
- Layout simples e organizado

Funcionalidades como:

- Sistema de pedidos
